Lenguaje R

R es un lenguaje y entorno de programación para análisis estadístico y gráfico. Es un dialecto libre del lenguaje S, desarrollado por Robert Gentleman y Ross Ihaka del Departamento de Estadística de la Universidad de Auckland en 1993.

Ventajas del lenguaje R

  • Proporciona un amplio numero de herramientas para estadísticas.
  • Permite generar gŕaficos de alta calidad usando sus funciones de fraficación.
  • Se puede usar tanto como herramienta de cálculo numérico y a la vez ser útil para la minería de datos.

Información extraída de: https://www.genbeta.com/desarrollo/r-un-lenguaje-y-entorno-de-programacion-para-analisis-estadistico

Variables y tipos de datos

R soporta tres tipos básicos de datos: Numericos, caracteres, cadenas de caracteres y boleanos. Estos datos se pueden almacenar en variables.

Asignación de variables

Para asignar variables se utiliza la siguiente sixtaxis:

a <- 4.6
b <- "Hola"
c <- TRUE

Vectores y matrices

Son dos de los tipos de datos compuestos que soporta R.

  • Vectores: Un vector es una concatenación de una colección de datos del mismo tipo. Cada dato recibe un índice según el orden en el que se concatenó (comenzando por 1). Para crear un vector usamos la función c(). Para crear un vector que consista en una serie de número consecutivos se utiliza el operador :. 
    • datos.consecutivos <- 1:10
datos <- c(2,1,3,-1,10,0,0,1)

    Podemos acceder a un dato utilizando su índice. Para acceder a un subconjunto de datos de un vector podemos utilizar otro vector con los índices correspondientes. Por último, podemos acceder a un subconjunto de datos de un vector utilizando un vector lógico. En este caso, se devolverán los datos que corresponden a TRUE en el vector lógico correspondiente. Por ejemplo: 

    datos[1]
datos[c(1,3,5)]
datos > 1
datos[datos > 1]
  • Matrices: Una matriz es una colección de datos del mismo tipo que tiene dos dimensiones. Para crear una matriz usamos la función matrix. Los elementos se proporcionan en un vector y se colocan por columnas: 
    • M <- matrix(1:10, nrow = 2, ncol = 5)
M

    Si queremos que se rellene por filas es necesario especificar el argumento byrow = TRUE: 

    M <- matrix(1:10, nrow = 2, ncol = 5, byrow=TRUE)
M

    La función dim devuelve las dimensiones de una matriz.

    La función nrow devuelve el número de filas.

    La función ncol devuelve el número de columnas:

    dim(M)
nrow(M)
ncol(M)

    Podemos asignar nombres a las filas y las columnas de una matriz: 

    rownames(M) <- c("fila1", "fila2")
colnames(M) <- c("columna1", "columna2", "columna3", "columna4", "columna5")
M

    Podemos seleccionar un elemento por su nombre:

    M[1,2]
M["fila2","columna2"]

    Podemos extraer filas y columnas enteras como vectores de matrices utilizando sólo el primer o segundo índice respectivamente y dejando el segundo en blanco: 

    M[1,]
M[,"columna3"]

Operadores aritmético-lógicos

  • Suma: +
  • Resta: -
  • Multiplicación: *
  • División: /
  • Potencia: ^
a <- 4
b <- 3
suma <- a + b
diferencia <- a – b
producto <- a * b
division <- a / b
potencia <- a ^ b

Las operaciones de comparación en R se escriben con los siguientes operadores:

  • Mayor que: >
  • Mayor o igual que: >=
  • Menor que: <
  • Menor o igual que: <=
  • Igual a: ==
  • Distinto de: !=

Estas operaciones devuelven true o false

a > 5
b >= 3
a < b
a <= 10
a == 5
b != a

Las operaciones lógicas en R se escriben con los siguientes operadores:

  • Conjunción (y): && o &
  • Disyunción (o): || o |
  • Negación (no): !
(a > 5) && (b > 2)
(a > 5) || (b > 2)
!(a > 5)

Qué se necesita para utilizar el lenguaje R

Para utilizar el lenguaje R se puede instalar RStudio, un programa disponible de forma gratuita para Windows, Mac y GNU/Linux. Se descargar aquí.

Ejemplo

Se quiere graficar la función coseno en el intervalo [0,2Π]

        x=seq(0,2*pi,length=100)
        plot(x,cos(x),type="l",col="blue",lwd=3)

En la primera linea se crea un vector que se usará con la función. En la segunda línea se grafica: la función plot() recibe varios parametros. El primero indica que variable se va a graficar, el segundo de donde va a obtener los valores, el tercer indica que graficará uniendo los puntos mediante líneas, el cuarto indica el color y el ultimo indica el grueso de la línea.
